# Afficionado Coffee Roasters - Product Optimization & Revenue Contribution Analysis

## Project Overview
This project analyzes product performance and revenue contribution for Afficionado Coffee Roasters.

## Objectives
- Identify top-selling products
- Analyze revenue contribution
- Perform category-level analysis
- Conduct Pareto (80/20) analysis
- Build an interactive Streamlit dashboard

## Tools Used
- Python
- Pandas
- Matplotlib
- Streamlit
- VS Code

## Key Findings
- Total Revenue: $698,812.33
- Coffee is the highest revenue-generating category
- Earl Grey Rg is the top-selling product
- Sustainably Grown Organic Lg is the highest revenue-generating product

## Dashboard Features
- KPI Cards
- Revenue Analysis
- Product Ranking
- Category Performance
- Interactive Visualizations
